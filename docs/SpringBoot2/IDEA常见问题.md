## 还原New window / Current windows

当打开新项目的时候,会询问是NewWindows还是当前窗口,如果不小心记录了,可在设置里搜索 ```System Settings```

## Spring Boot Application in default package

请把 ```xxApplication``` 文件放在 ```package``` 目录中,不要放在JAVA根目录
